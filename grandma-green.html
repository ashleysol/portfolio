Role

Lead UI Engineer

Skills & Tools

Grandma Green is a virtual pet and farming simulation game with over 60,000 downloads and a 4.2 rating on the App Store. Grandma Green is a beautiful game that has grown a hefty following spread out across TikTok, YouTube, Twitter, and more!

My experience at Grandma Green was truly amazing. Working on a UI-heavy game reinforced the importance of creating a UI that is aesthetic, communicates clearly, and creates bugless interactions. I was surrounded by a team that, like me, was truly passionate about the work we were doing and our users' experience. I was able to develop my skills as an engineer and be a part of creating something I am very proud of. Here is a quick list of what I did:

  • Created over 50 user interfaces using Unity’s version of HTML and CSS
  • Worked closely with designers and artists to transform mockups into code
  • Achieved responsive UI across all mobile device screens
  • Collaborated with engineers to seamlessly bridge back-end systems and data with user interfaces
  • Produced high-quality animations using C# and Unity UI Toolkit
  • Participated in Scrum and completed 5+ tickets a week
  • Debugged and tested code before weekly prototypes
